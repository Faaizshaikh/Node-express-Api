Node.js Express API

A complete RESTful API for task management with Express.js and Node.js

Fast & Lightweight

Built with Express.js for high performance and minimal overhead. Perfect for small to medium applications.

JSON Data Storage

Uses a JSON file for data storage, making it easy to set up and use without a database server.

RESTful Design

Follows REST principles with proper HTTP methods and status codes for all endpoints.

API Endpoints

GET /api/tasks
Retrieve all tasks from the system
POST /api/tasks
Create a new task with a title and completed status
PUT /api/tasks/:id
Update an existing task by its ID
DELETE /api/tasks/:id
Delete a task from the system by its ID
